# Browser Research

If the user asks to browse, open, click, snapshot, inspect, or extract from a page, use browser automation.

Never replace a browser task with a native web search summary. Search may only be used to discover the URL when the user did not provide one.

Ask for the URL when the task requires a specific page and no URL is available.
